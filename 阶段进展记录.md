### 5.1 当前结果概览
按当前 `comparison_overall.csv`：

- `total_field_highpass`：峰值响应最强，但波动最大，属于最激进基线
- `wavelet_denoise`：较保守，平滑明显，异常峰值也被压低
- `lowrank_sparse`：已成功接入，表现出结构背景分离能力，在部分复杂背景轨迹上具有优势
- `emd`：在背景净化与异常保留之间较平衡
- `vmd`：当前参数下过于保守，疑似净化过度
- `obf`：更像低阶背景拟合对照组
- `gradient_obf`：在梯度通道上对局部异常更敏感

当前 7 个方法的总体摘要为：

- `total_field_highpass`：`mean_residual_rms = 3.2922`，`mean_response_mean = 5.1445`
- `wavelet_denoise`：`mean_residual_rms = 2.0578`，`mean_response_mean = 0.8850`
- `lowrank_sparse`：`mean_residual_rms = 1.8969`，`mean_response_mean = 0.9677`
- `emd`：`mean_residual_rms = 0.7808`，`mean_response_mean = 1.0373`
- `vmd`：`mean_residual_rms = 0.0283`，`mean_response_mean = 0.8056`
- `obf`：`mean_residual_rms = 5.6065`，`mean_response_mean = 3.2845`
- `gradient_obf`：`mean_residual_rms = 0.9830`，`mean_response_mean = 2.2124`

关键输出：

- `outputs/tables/decomposition/comparison_per_file.csv`
